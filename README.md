This project is a **personal, non-commercial reference tool** created for **private use only**.  
It is **not affiliated with, endorsed by, or connected to GONE Fishing or GONE Studios** in any way.

All game names, images, species names, bait names, and related assets are the intellectual property of **GONE Studios** and are used here **solely for personal reference and educational purposes**.

## Purpose of This Project

The purpose of this repository is to:

- Create **locally stored reference data** for personal analysis and planning  
- Mirror the **structure** of the official GONE Fishing encyclopedias for **offline use**
- Experiment with **data parsing, visualization, and tooling** without interacting with live services

This project **does not**:
- Modify or interfere with the GONE Fishing website
- Submit data to the leaderboard
- Automate gameplay
- Provide competitive advantages or cheats
- Replace or replicate official services
